# FW-Msg

## Beschreibung
FW-Msg ist ein Tool für die Verwaltung von Freiwilligenarbeit. Es ermöglicht die Verwaltung von Freiwilligen, Projekten und Aufgaben.

## Benutzung
### Installation
1. Installiere Python 3.8 oder höher
2. Installiere Django 3.0 oder höher
3. Installiere das Projekt
```bash
git clone
cd FW-Msg
pip install -r requirements.txt
```

### Starten
```bash
python manage.py runserver
```

## Dokumentation
